🚀 Rocket Performance Calculator

Overview

The Rocket Performance Calculator is a Python-based tool developed to estimate the flight performance of Level 1 and Level 2 high-power rockets.

The calculator uses user-defined rocket and motor parameters to predict key flight characteristics, providing a simple engineering assessment prior to launch.

Features

* Weight Calculation
* Thrust-to-Weight Ratio (TWR)
* Initial Acceleration
* Rail Exit Velocity
* Burnout Velocity
* Apogee Estimation
* Time to Apogee
* Maximum Mach Number
* Dynamic Pressure (Max-Q)

Inputs

Rocket Parameters

* Liftoff Mass (kg)
* Propellant Mass (kg)

Motor Parameters

* Motor Designation
* Average Thrust (N)
* Burn Time (s)

Atmospheric Conditions

* Air Density (kg/m³)

Launch Conditions

* Launch Rail Length (m)
* Launch Angle (deg)

Engineering Concepts Applied

* Newton’s Second Law
* Thrust-to-Weight Analysis
* Rocket Flight Mechanics
* Kinematic Equations
* Conservation of Energy
* Mach Number Estimation
* Dynamic Pressure Analysis
